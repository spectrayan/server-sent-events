# Security Policy

We take the security of Spectrayan SSE seriously.

- Report vulnerabilities privately to: support@spectrayan.com
- Please include steps to reproduce and any relevant logs or PoCs.
- We will acknowledge receipt within 3 business days and provide a timeline for a fix after triage.
- Do not file public GitHub issues for sensitive security reports.

## Supported versions
This is an active OSS project. We generally patch the latest minor for each maintained major. Security fixes may be backported case‑by‑case.

## Responsible disclosure
If you believe you have found a critical issue, please give us reasonable time to investigate and release a patch before any public disclosure.
