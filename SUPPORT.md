# Support

If you need help using Spectrayan SSE or have questions:

- Open a GitHub issue using one of our templates (bug report or feature request)
- For private matters or security concerns, email: support@spectrayan.com

Please include:
- What you tried and what you expected
- Steps to reproduce
- Environment details (OS, Node.js, npm, Java/Maven, Angular/Spring versions)
- Logs or stack traces where relevant
